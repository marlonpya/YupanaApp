# Yupana

App móvil (Android + iOS, Kotlin Multiplatform) para revendedores de suscripciones de
streaming: organiza cuentas, perfiles, clientes y recordatorios de vencimiento de cobro.

- **[`CLAUDE.md`](CLAUDE.md)** — documento maestro: producto, stack, arquitectura,
  backend Supabase, identidad visual y reglas de trabajo. Léelo antes de tocar código.
- **[`MODULES.md`](MODULES.md)** — mapa de módulos Gradle, existentes y planeados.
- **[`docs/DESARROLLO.md`](docs/DESARROLLO.md)** — cómo compilar y verificar (incluye el
  setup específico para WSL).
